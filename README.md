# Expense & Budget Tracker Backend API

A clean and scalable **Expense & Budget Tracker Backend** built using **Node.js, Express, and TypeScript**, following **Object-Oriented Programming (OOP)** principles and a layered architecture.

This project allows users to manage daily expenses with support for **CRUD operations, search, filtering, sorting, and pagination**, and is designed to be extended with **authentication, budgeting, and database integration**.
